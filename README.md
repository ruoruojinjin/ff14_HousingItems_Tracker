# 🏠 FF14 Housing Items Tracker ✨ 房屋家具清單工具
>　一個為《Final Fantasy XIV》房屋裝潢玩家設計的小工具，用來整理家具清單、規劃採購流程，讓裝潢更有效率。

### 👉 [線上使用](https://ruoruojinjin.github.io/ff14_HousingItems_Tracker)
<br>

## ✨ 功能特色
📦 家具清單整理與統計：集中管理你設計中所需的家具。

🔢 自動計算所需數量：避免遺漏或重複購買。

📊 匯出 CSV：方便帶去市場購買（可用 Excel 打開）。

🔍 快速查找與管理家具：強大的搜尋與篩選功能。

⚡ 輕量快速：免安裝即可直接在瀏覽器使用。

## 🛠️ 本地開發 (Local Development)
如果你想在本地跑這個專案：
```
git clone https://github.com/ruoruojinjin/ff14_HousingItems_Tracker.git

npm install

npm run dev
```

## 🚀 使用方式 Usage
開啟工具：訪問 線上版。

輸入你的傢俱名稱。

自動整理所需家具與數量。

匯出 CSV，帶去市場購買 🛒。
